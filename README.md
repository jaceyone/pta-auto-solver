# pta-auto-solver
基于 Playwright MCP 实现 PTA 拼题 A 全自动刷题，自动登录、抓题、答题提交，兼容选择判断、编程题，适配 CodeMirror6 编辑器，仅登录、选题库、图片难题三处需人工确认，多浏览器内核适配，不保存账号密码
模块	说明
项目简介	面向 PTA 拼题 A 的自动化刷题 Claude Code 技能，实现页面全自动解题提交
功能亮点	登录、抓取题目、作答、提交全自动化；适配 CodeMirror 6 代码注入；批量处理单选 / 判断
前置依赖	Claude Code、Playwright MCP
安装方法	1. git clone 项目2. 移入 Claude Code skills 目录3. 或通过 /skill 命令安装
使用流程	1. 工具自动部署浏览器内核2. 手动输入账号密码登录 PTA3. 指定刷题分类批量抓取题目4. 自动答题提交；图片无法识别则交由用户处理
目录说明	内置中英文配套脚本、选择器配置、代码注入工具脚本，分文件夹存放
安全声明	不会保存、泄露账号密码；无法解答的题目如实提示，不编造虚假答案
贡献指南	欢迎提交 PR 优化功能与适配逻辑
